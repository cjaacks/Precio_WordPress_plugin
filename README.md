Precio_WordPress_plugin
=======================

Manejo sencillo de precios de productos para WordPress.

Muestra el precio grabado en un custom field llamado ‘precio’ de un post o página. Para ello busca la cadena ‘[precio]‘ en el cuerpo del post. Si el valor del precio en el custom field es -1, muestra una leyenda de ‘producto fuera de stock’. Permite efectuar una búsqueda por rango de precios. Si desea usar centavos, utilize el punto en los decimales (no la coma) al cargar los precios en los customs fields. Se usa {?php Mostrar_Precio(); ?} desde el template para mostrar el precio en una página o post y {?php Mostrar_Form_Rango(); ?} para desplegar el formulario de búsqueda por precio para el usuario.
